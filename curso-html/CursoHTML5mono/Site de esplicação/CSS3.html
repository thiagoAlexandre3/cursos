O que é CSS3

Como dito anteriormente, O CSS3 cuida da estrutura do site, ele realiza funções que deixa o site de uma forma mais bonita e elegante, ou seja, ele trabalha com o design do layout da página. A principal função do CSS3 é abolir as imagens de plano/Background de fundo, bordas arredondadas, apresentar transições e efeitos para criar animações de vários tipos, como um simples relógio de ponteiros. É chamado de CSS 3, pois foi aprimorado como o HTML, antes era chamado apenas de CSS e passou a se chamar CSS3, pelas alterações e dições feitas nesta importante linguagem de formatação. Esta linguagem atualmente vem substituindo algumas Tags de HTML, para um padrão mais tecnico, que deixa o código melhor, falando semânticamente.
