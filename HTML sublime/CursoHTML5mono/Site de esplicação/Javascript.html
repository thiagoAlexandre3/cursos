O que é JavaScript

O JavaScript, também chamado de JS, diferente do HTML e o CSS, é uma linguagem de programação.É atualmente a principal linguagem para programação client-side em navegadores web. Quando você acessa a uma página na internet, a página acessada é pega na parte Client-side, e é alterada para você na parte do servidor, esses conceitos são usados quando se estuda essas linguagem, tanto de formatação, como de programação. O JavaScript cuida da parte interativa do site, como por exemplo, um botão que quando você apenta, realiza uma ação. O exemplpo dado é bem simples de se fazer, com o JavaScript dizemos, se o batão for precionado, então, faça ..., se não, faça ... E é assim que se programa em JavaScript.
