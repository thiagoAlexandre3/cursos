O que é HTML5

HTML5, é uma linguagem de formatação, ultilizada pelo World Wide Web, e também é muito importante para a construção de sites, pois ela monta a semântica da página em que nos acessamos em nosso cotidano. Por que eu disse a palavra semântica, pois o HTML cuida da parte "escrita" do código, temos outras linguagem que cuidam em deixar o site melhor, além do html, como por exemplo, o CSS3 e o JavaScript. Voltando pro HTML, ele foi originalmente proposto por Opera Software, e após foi desenvolvido, ou melhor, edificado por Web Hypertext Application Technology Working Group, que trouxe novas ideias e aprimorou o HTML para o que copnheçemos hoje.

O que é CSS3

Como dito anteriormente, O CSS3 cuida da estrutura do site, ele realiza funções que deixa o site de uma forma mais bonita e elegante, ou seja, ele trabalha com o design do layout da página. A principal função do CSS3 é abolir as imagens de plano/Background de fundo, bordas arredondadas, apresentar transições e efeitos para criar animações de vários tipos, como um simples relógio de ponteiros. É chamado de CSS 3, pois foi aprimorado como o HTML, antes era chamado apenas de CSS e passou a se chamar CSS3, pelas alterações e dições feitas nesta importante linguagem de formatação. Esta linguagem atualmente vem substituindo algumas Tags de HTML, para um padrão mais tecnico, que deixa o código melhor, falando semânticamente.

O que é JavaScript

O JavaScript, também chamado de JS, diferente do HTML e o CSS, é uma linguagem de programação.É atualmente a principal linguagem para programação client-side em navegadores web. Quando você acessa a uma página na internet, a página acessada é pega na parte Client-side, e é alterada para você na parte do servidor, esses conceitos são usados quando se estuda essas linguagem, tanto de formatação, como de programação. O JavaScript cuida da parte interativa do site, como por exemplo, um botão que quando você apenta, realiza uma ação. O exemplpo dado é bem simples de se fazer, com o JavaScript dizemos, se o batão for precionado, então, faça ..., se não, faça ... E é assim que se programa em JavaScript.
